# Biblia de arte — NEON EXODUS 2087
**Iria Cromo · Dirección de arte · Estudio Siete//Siete**

## Tres reglas
1. Neón sobre negro-violeta. Nada de grises tibios.
2. La rejilla siempre corre hacia el horizonte. El mundo nunca está quieto.
3. El sol tiene franjas. Siempre.

## Paleta maestra
| Uso | Color |
|---|---|
| Fondo | `#0a0014` |
| Magenta (peligro que persigue) | `#ff2bd6` |
| Cian (jugadora, aliados, UI) | `#19e6ff` |
| Amarillo (puntuación, blindaje) | `#ffd319` |
| Verde (vida, escudo, cloacas) | `#2bff88` |
| Rojo (jefe, daño) | `#ff3b3b` |
| Violeta (lo espectral) | `#8a2bff` |

Cada sector reescribe la paleta del cielo, el sol y la rejilla para que el
progreso se *sienta* antes de leerse: magenta callejero → ámbar de mercado →
verde de cloaca → rojo corporativo → blanco-violeta del núcleo.

## Lenguaje de formas
- **Jugadora:** flecha cian con cabina magenta. Siempre el trazo más limpio de la pantalla.
- **Drone:** triángulo magenta. Punta = intención.
- **Spinner:** cuadrado doble girando. Lo geométrico es burócrata.
- **Tank:** hexágono concéntrico. La masa se lee desde lejos.
- **Wraith:** rombo translúcido que late. Si parpadea, no está del todo aquí.
- **Splitter:** círculo con núcleo triangular: lleva crías dentro y se nota.
- **Turret:** pentágono con cañón. Lo estático apunta.
- **M.A.D.R.E.:** tres anillos contrarrotando alrededor de un ojo. Una catedral de geometría hostil.

## Atmósfera
- CRT por CSS: scanlines, viñeta y flicker a 8 Hz. Desactivable (accesibilidad).
- Skyline procedural con ventanas encendidas; cada sector genera su silueta con semilla propia.
- Partículas aditivas (`lighter`): toda muerte es un pequeño espectáculo de feria.
- El texto vive en el DOM, no en el canvas: el neón puede ser borroso, la palabra no.
